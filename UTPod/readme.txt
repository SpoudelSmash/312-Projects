Project UTPod:
Implementation of a song library in cpp

Included files: zip (UTPodDriver.cpp, UTPod.h/cpp, Song.h/cpp, makefile)

Compile & Run(can be done in a Linux environment):
    1. Unzip the files into the same directory
    2. Execute "make" command. This will create a executable named "math_stuff"
    3. Run the executable with "./UTPod". You are now running the program.
