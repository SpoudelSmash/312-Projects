Project BST312
Implementation of a binary search tree in cpp

Included files: readme.txt, BST312.h

The header file BST312.h contains a class for a binary search tree with the functions:

makeEmpty
--Removes all the items from the BST.

isEmpty
--Checks to see if there are any items in the BST.

isFull
--Determines if you have any more room to add items to the BST

insertItem
--Adds newItem to the BST.

deleteItem
--Removes target from BST.

countNodes
--Counts the number of nodes in the BST

preOrderTraversal
--Returns the preOrder (Node, Left, Right) as a vector of ItemTypes

inOrderTraversal
--Returns the inOrder (Left, Node, Right) as a vector of ItemTypes

postOrderTraversal
--returns the postOrder (Left, Right, Node) as a vector of ItemTypes


isItemInTree
--Determines if a given Item is in tree.
